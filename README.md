# Spatiotemporal Demand Forecasting and Reliability Analysis for Toronto Bike Share

## Overview

This repository contains the complete implementation of my Master of Data Science and Analytics Major Research Project (MRP) at Toronto Metropolitan University.

The objective of this project is to study bike-sharing demand patterns, analyze weather impact on ridership behavior, detect station imbalance conditions, and build future predictive models capable of improving operational reliability within Toronto Bike Share.

## Project Objectives

* Analyze temporal demand behavior in Toronto Bike Share
* Study weather impact on bike-sharing demand
* Detect empty-risk and full-risk stations
* Perform operational reliability analysis
* Build machine learning demand forecasting models
* Develop future proactive bike rebalancing framework

## Dataset

* Toronto Bike Share Operational Data (2021–2025)
* Toronto Historical Climate Dataset (2021–2025)

## Project Structure

/data → Raw and cleaned datasets
/notebooks → Google Colab for EDA and preprocessing
/visualizations → Graphs and charts generated during analysis
/models → Machine learning forecasting models
